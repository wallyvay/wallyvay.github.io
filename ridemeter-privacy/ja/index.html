プライバシーポリシー

RideMeter
最終更新日：2026年5月7日

RideMeter はお客様のプライバシーを重視しています。本プライバシーポリシーでは、当社がお客様の情報をどのように収集、利用、保護するかを説明します。

概要：すべてのライドデータはお客様のデバイス内にのみ保存され、個人情報を収集してサーバーへアップロードすることはありません。

1. 情報の収集

主要機能を提供するため、RideMeter は以下の権限を必要とします：

  • 位置情報（常に/使用中）：ライドナビゲーション、GPS による速度計算、走行ルートの記録に使用します。位置情報の権限は iOS のシステム設定でいつでも変更できます。
  • ライドデータ：リアルタイム速度、距離、走行時間、平均速度、ルート軌跡が含まれます。

2. データの保存

RideMeter はローカルファーストの設計方針に基づいています：

  • すべてのライドデータはお客様の iPhone 端末内にのみ保存されます
  • 個人を特定できる情報を外部サーバーへアップロードすることはありません
  • お客様のデータを第三者と共有することはありません
  • 本アプリは第三者の分析サービスや広告 SDK を使用しません
  • 本アプリはアカウント作成、登録、ログインを必要としません

3. 位置情報の利用

位置情報は以下の目的にのみ使用します：

  • ライドナビゲーションルートの表示
  • リアルタイム速度と走行距離の計算
  • 走行軌跡の記録

位置データはアプリが前面で使用されている間のみ収集されます。アプリを閉じている場合やバックグラウンドで動作している場合、位置情報を継続的に取得することはありません。

4. データの共有

お客様が共有機能を能動的に使用する場合：

  • アプリはライドデータを含む画像を生成します
  • その画像を WeChat、Weibo、モーメンツなどのソーシャルプラットフォームへ共有するかを選択できます
  • これはお客様が能動的に開始する操作であり、アプリがデータを自動的に共有することはありません

5. データの削除

お客様はいつでも以下を行えます：

  • アプリ内で単一またはすべてのライド履歴を削除する
  • アプリをアンインストールしてすべてのローカルデータを消去する
  • iOS のシステム設定で位置情報権限をオフにし、位置データの収集を停止する

6. 子どものプライバシー

本アプリは13歳未満の子どもを対象としていません。当社は子どもの個人情報を故意に収集しません。

7. ポリシーの変更

本プライバシーポリシーに変更がある場合、アプリ内で通知し、本ページの「最終更新日」を更新します。

8. お問い合わせ

本プライバシーポリシーについてご質問や懸念がある場合は、以下の方法でお問い合わせください：

  • GitHub Issues：wallyvay/ridemeter
  • または本プロジェクトのリポジトリで Issue を作成してください
