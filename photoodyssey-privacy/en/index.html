Privacy Policy

Last updated: June 4, 2026

PhotoOdyssey respects your privacy. This policy explains how the app handles photos, library indexes, natural-language search configuration, and folders you choose to access.

In short: PhotoOdyssey does not require an account. Selected photo folders, thumbnails, indexes, and analysis results stay on your Mac by default. The app does not upload your original photos to developer servers.

1. Information We Use

To provide browsing, indexing, filtering, and search features, PhotoOdyssey reads photos in folders you authorize, file paths, file names, capture time, image dimensions, formats, color or visual features, thumbnails, and metadata available from the system.

2. Local Indexing and Storage

PhotoOdyssey builds a local library database, thumbnail cache, and analysis results so you can browse and find photos quickly. The library database references original file paths and does not copy or upload original photos. You can delete the library file or uninstall the app to remove local app data.

3. Folder Permissions

In the macOS sandbox, PhotoOdyssey accesses only folders or library files you authorize through the system file picker. You can stop using a library through system settings or app workflows at any time.

4. Natural-Language Search and Model Configuration

PhotoOdyssey lets you configure your own large language model service for natural-language photo search. Whether to enable it, which service to use, and what search requests to send are configured and triggered by you. Third-party model services process request content under their own privacy policies. PhotoOdyssey does not operate developer servers for receiving your photos.

5. Data Collection

PhotoOdyssey does not create user accounts and does not collect names, email addresses, contacts, precise location, or personally identifiable information.

6. Children's Privacy

PhotoOdyssey is not intended for children under 13. We do not knowingly collect personal information from children.
